Ну и потом просто перейти по адресу (обычно это http://127.0.0.1:8000/), который выводится в терминале.  
Но если есть PyCharm, то можно просто открыть в нем проект и ввести следующие команды:  
cd .\music_project\  
python manage.py runserver  
И так же перейти по адресу.
