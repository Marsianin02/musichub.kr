Я делал все в PyCharm, поэтому способы запуска без него несколько костыльны.
Самое банальное что я смог найти, это открыть папку, в которой располагается "python.exe", в терминале и далее просто ввести следующее (ну или скопировать):  
.\python.exe "тут весь путь до этой директории\music_project\manage.py" runserver  
Ну и потом просто перейти по адресу (обычно это http://127.0.0.1:8000/), который выводится в терминале.  
Но если есть PyCharm, то можно просто открыть в нем проект и ввести следующие команды:  
cd .\music_project\  
python manage.py runserver  
И так же перейти по адресу.
